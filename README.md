# monty-hall-sim

This is a simulation of the Monty-Hall problem.

## Installation

Clone, and create a JAR file with "lein uberjar"

## Usage

    $ java -jar monty-hall-sim-0.1.0-standalone.jar times

where 'times' is the number of times to run the game.
